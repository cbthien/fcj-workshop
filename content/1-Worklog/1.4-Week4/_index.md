### Week 4 Objectives:

* Understand how to launch and manage both Windows and Linux EC2 instances.
* Learn how to deploy applications on EC2 (Node.js & AWS User Management App).
* Understand IAM governance and cost usage governance in EC2 environments.
* Learn how applications securely access AWS services using IAM roles instead of access keys.
* Practice attaching IAM roles to EC2 and testing application access.

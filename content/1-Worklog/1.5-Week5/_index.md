### Week 5 Objectives:

* Learn how to create and work with an AWS Cloud9 development environment.
* Practice using Cloud9 IDE features and AWS CLI inside Cloud9.
* Understand Amazon S3 fundamentals and how to host a static website on S3.
* Configure S3 bucket security settings including Public Access Block and object permissions.
* Implement S3 features such as Versioning, Object Movement, and Cross-Region Replication.
* Learn how to integrate S3 with CloudFront for accelerating static websites.
